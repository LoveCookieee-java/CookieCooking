# 🍳 CookieCooking

**CookieCooking** là một hệ thống nấu ăn nâng cao dành cho Minecraft server chạy nền tảng **PySpigot**. Plugin mang đến trải nghiệm nhập vai nấu nướng cực kỳ chân thực, trực quan thông qua Hologram (Item Display) và GUI.

---

## 🌟 Tính Năng Nổi Bật

- **4 Trạm Nấu Ăn Độc Lập:** 
  - 🔪 **Thớt (Chopping Board):** Thái nguyên liệu bằng Dao. Có rủi ro đứt tay nếu không cẩn thận!
  - 🍳 **Chảo (Wok):** Đảo thức ăn liên tục với 3 cấp độ lửa khác nhau. Thức ăn có thể bị cháy khét hoặc sống nếu nấu sai cách.
  - ⚙️ **Cối Xay (Grinder):** Xay nhuyễn nguyên liệu theo thời gian thực với hiệu ứng và âm thanh đẹp mắt.
  - 🫕 **Nồi Hấp (Steamer):** Hệ thống hấp phức tạp yêu cầu duy trì Củi (Fuel) và Nước (Moisture).
- **Hỗ Trợ Item Tùy Chỉnh:** Tương thích hoàn toàn với các vật phẩm custom từ `MMOItems`, `CraftEngine`, `NeigeItems` và dĩ nhiên là Vanilla Minecraft.
- **Hệ Thống Phân Quyền:** Quản lý quyền hạn cho từng thao tác và từng công thức riêng biệt (`cookiecooking.*`).

## 📦 Yêu Cầu Hệ Thống

- **Phiên bản Minecraft:** `1.21` đến `1.21.11`
- **Nền tảng:** Server hỗ trợ PySpigot
- **Dependencies:** PlaceholderAPI

## 📖 Hướng Dẫn Sử Dụng (Wiki)

Bạn có thể tìm thấy toàn bộ cách thức cài đặt, phân quyền, các lệnh và cách tự do sáng tạo Recipe trong tài liệu **[CookieCooking_Wiki.md](CookieCooking_Wiki.md)** đính kèm trong repository này.

## 🛠️ Cài Đặt

1. Đảm bảo server của bạn đã cài đặt **PySpigot**.
2. Đặt file `CookieCooking.py` và toàn bộ thư mục `CookieCooking` vào đúng vị trí thư mục script của PySpigot.
3. Khởi động server hoặc chạy lệnh `/cooking reload` trong game để tải cấu hình và công thức.
